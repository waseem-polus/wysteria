## API Reference

##### Chip

The main chip component

| Prop                   | Type                                  |    Default    |
| ---------------------- | ------------------------------------- | :-----------: |
| <Prop>variant</Prop>   | `filled` `ghost` `outline` `text`     |   `filled`    |
| <Prop>action</Prop>    | `progressive` `destructive` `neutral` | `progressive` |
| <Prop>disabled</Prop>  | `boolean`                             |    `false`    |
| <Prop>className</Prop> | `string`                              |       -       |

##### ChipAction

An action button to interact with the chip. Accepts the same props as a [`Button`](/components/button)
