# Toggle

A two-state button that can be either on or off.

<ComponentPreview>
    <Toggle size="icon" offChildren={<BellOff size={20} />}>
        <BellRing size={20} />
    </Toggle>
</ComponentPreview>
```jsx
<Toggle offChildren={<BellOff size={20} />}>
    <BellRing size={20} />
</Toggle>
```

## API Reference

Toggle accepts any [Button](/components/button) prop in addition to its own

| Prop                      | Type         | Default |
| ------------------------- | ------------ | :-----: |
| <Prop>initialValue</Prop> | `boolean`    | `true`  |
| <Prop>onChange</Prop>     | `() => void` |    -    |
| <Prop>offChildren</Prop>  |              |         |
