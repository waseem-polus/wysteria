## Examples

### External

An <Link target="_blank" href="https://lucide.dev/icons/external-link" hideIcon>external link</Link> icon is appended if the link starts with `http[s]://`

  <TabGroup>
    <Tab label="Preview">
      <Container paddingVer="md" gap="sm" justify="space-evenly" align="center" direction="row" style={{minHeight: "5rem"}}>
        <Link href="https://www.wysteria.dev">Visit Us</Link>

    </Tab>

    <Tab label="Code">

## API Reference

Supports any props the <Link target="_blank" href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a" hideIcon>`<a>` element</Link> accepts.

| Prop                                      | Type                                             | Default |
| ----------------------------------------- | ------------------------------------------------ | :-----: |
| <Prop>href</Prop>                         | `string`                                         |    -    |
| <Prop>target</Prop>                       | `_self` `_blank` `_parent` `_top` `_unfencedTop` |    -    |
| <Prop hint="hideLinkIcon">hideIcon</Prop> | `boolean`                                        | `false` |
| <Prop>className</Prop>                    | `string`                                         |    -    |
