## API Reference

| Prop                      | Type                                                    |  Default  |
| ------------------------- | ------------------------------------------------------- | :-------: |
| <Prop>type</Prop>         | `text` `number` `email` `password` `tel` `url` `search` |  `text`   |
| <Prop>status</Prop>       | `neutral` `error` `success`                             | `neutral` |
| <Prop>placeholder</Prop>  | `string`                                                |     -     |
| <Prop>value</Prop>        | `string`                                                |     -     |
| <Prop>defaultValue</Prop> | `string`                                                |     -     |
| <Prop>onChange</Prop>     | `(e) => void`                                           |     -     |
