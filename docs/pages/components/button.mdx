## API Reference

| Prop                   | <div className="w-full">Type</div>                                        |    Default    |
| ---------------------- | ------------------------------------------- | :-----------: |
| <Prop>variant</Prop>   | `filled` \| `outlined` \| `text`            |   `filled`    |
| <Prop>action</Prop>    | `progressive` \| `destructive` \| `neutral` | `progressive` |
| <Prop>size</Prop>      | `normal` \| `icon`                          |   `normal`    |
| <Prop>disabled</Prop>  | `boolean`                                   |    `false`    |
| <Prop>onClick</Prop>   | `() => void`                                |       -       |
| <Prop>className</Prop> | `string`                                    |       -       |
