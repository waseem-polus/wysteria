# Chip

Interactive badges used to represent user-generated content.

<ComponentPreview>
    <Chip>
        Chip
        <ChipAction />
    </Chip>
</ComponentPreview>

```jsx
<Chip>
    Chip
    <ChipAction />
</Chip>
```

## API Reference

| Prop                   | Type                                  |    Default    |
| ---------------------- | ------------------------------------- | :-----------: |
| <Prop>variant</Prop>   | `filled` `outline`                    |   `filled`    |
| <Prop>action</Prop>    | `progressive` `destructive` `neutral` | `progressive` |
| <Prop>className</Prop> | `string`                              |       -       |
